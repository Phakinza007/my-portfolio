01

Brief ชัด

02

ลำดับสายตา

03

Motion พอดี

ใช้การเคลื่อนไหวเพื่อบอกสถานะและเน้นจังหวะสำคัญ โดยไม่แย่งความสนใจจากเนื้อหา

04

Mobile พร้อม

ตรวจเมนู ปุ่ม ฟอร์ม และระยะห่างบนหน้าจอเล็กก่อนเอาหน้าเว็บไปใส่พอร์ต

Roadmap

  • HTML5
  • CSS Grid

1

วางโครงสร้างแบบ Semantic

แบ่ง header, main, section, article, aside, footer ให้ถูกหน้าที่ เพื่อให้เว็บอ่านง่ายและดูเป็นมืออาชีพ

2

จัด Layout หลายชั้น

ใช้ grid และ flexbox ร่วมกัน เช่น hero สองฝั่ง, feature cards, pricing cards และ contact form

3

เพิ่ม Interaction ด้วย CSS

ใส่ hover, transition, animated progress, floating card และ pulse status โดยไม่ต้องใช้ JavaScript

4

ปรับ Responsive ให้ครบ

เขียน media queries หลาย breakpoint เพื่อให้หน้าเว็บใช้งานได้ทั้ง desktop, tablet และ mobile
