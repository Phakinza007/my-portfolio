Elasticshop Gaming
Top-Up Platform

A complete UI design system for an automated game top-up platform. Built with a "Reliable Pro-Gamer" brand personality — fast, trustworthy, and visually striking. Dark foundations, high-contrast yellow highlights, and precision red CTAs.

Figma Design System Mobile-First Thai / English Bilingual 4 Screen Flows
Screens

4 Core User Flows

From browsing games on the home screen to confirming a top-up and reviewing transaction history — every critical touchpoint designed for speed and trust.

Home Screen
Game catalog with skeleton loading state and gray navigation bar. Entry point for browsing available top-up games.
Top-Up Flow
3-step flow: Player Info → Package Selection → Payment. Yellow highlights for promotions, red CTA for confirmation.
Transaction Status
Post-payment feedback screen showing success, pending, and failed states with clear visual language and next actions.
Transaction History
Full transaction log with filterable status badges, game thumbnails, and a scannable list layout for quick review.

System

Design Principles

Built for the "Reliable Pro-Gamer" persona — frictionless, precise, and visually energetic.

⚡
One-Tap Feel
Interface stays out of the way. The user's goal is getting back in the game — every interaction is designed to feel instant.
🎯
Action Hierarchy
Red is reserved exclusively for high-priority CTAs (Buy Now, Confirm). Yellow for promotions and attention. Never mixed.
◼
Tonal Layers
Elevation through tonal contrast and 1px borders rather than heavy shadows. Keeps the UI feeling fast and flat-modern.
📐
8px Grid
All spacing driven by an 8px base unit. 12-column desktop, 2-column tablet, single-column mobile-first stack.
🌐
Bilingual UI
Prompt handles English display type. Noto Sans Thai handles functional labels — both tested for legibility at small sizes.
🔴
Selected States
Active package/payment cards get a 2px Primary Red or Yellow border with a checkmark. No elevation change needed.
